| Field | Value |

This tiny file lets `demo/run_demo` run **offline, with zero downloads**, a hard
requirement for every project (CLAUDE.md §8). It is not measured data — it is the
*definition* of a reaction network whose answer is known in closed form.

### Field meanings

| Token | Meaning | Units | Sample value |
|---|---|---|---|
| `k_prod` | transcription (production) rate of the source reaction `∅ → M` | molecules / time | `10` |
| `k_deg` | per-molecule degradation rate of `M → ∅` | 1 / time | `0.5` |
| `m0` | initial mRNA count | molecules (integer) | `0` |
| `t_end` | simulation horizon | time units | `50` |
| `n_traj` | number of independent SSA trajectories (ensemble size) | count | `256` |
| `base_seed` | RNG base seed (reproducibility) | integer | `20240611` |

### The model and its known answer

The sample encodes the canonical **birth–death gene-expression** model:

```
  R1:  ∅  ── k_prod ──►  M       (constant transcription)
  R2:  M  ── k_deg  ──►  ∅       (first-order degradation)
```

Its stationary distribution is **Poisson with mean `λ = k_prod / k_deg`**. For the
sample values, `λ = 10 / 0.5 = 20` molecules. The demo recovers this analytic mean
from the ensemble — a science check on top of CPU==GPU agreement (see THEORY §6).

Regenerate or resize the sample:

```bash
python scripts/make_synthetic.py                     # defaults (λ = 20, 256 trajectories)
python scripts/make_synthetic.py --k-prod 20 --k-deg 1 --n-traj 4096
```

## Full / real datasets

This project does not *require* a downloaded dataset — its input is a small model
specification. To study real stochastic models, fetch curated networks and adapt
`build_gene_network()` in `src/reference_cpu.cpp` to their stoichiometry:

- **BioModels Database** — curated stochastic **SBML** models.
  <https://www.ebi.ac.uk/biomodels> · License: CC0 for curated models (check each
  model's page). SBML parsing is left as an exercise; `scripts/download_data.*`
  prints instructions.
- **NIST Chemical Kinetics Database** — reaction rate constants.
  <https://kinetics.nist.gov>
- **DANDI** — single-molecule tracking datasets. <https://dandiarchive.org>
- **GEO** — smFISH single-cell gene-expression counts (for comparing simulated to
  measured distributions). <https://www.ncbi.nlm.nih.gov/geo/>

Respect every license. Credentialed sources are never bypassed by the download
scripts — they print registration instructions and links only (CLAUDE.md §8).

## Provenance & honesty

Everything under `sample/` is **synthetic** and labeled as such. No real
patient-derived or proprietary data is committed. The model's analytic mean is a
mathematical fact, not a clinical claim; this is educational study material only.

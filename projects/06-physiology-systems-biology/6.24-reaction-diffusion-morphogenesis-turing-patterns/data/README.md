There is no "dataset" to download for a Turing simulation — **the data *is* the
model configuration**, and the pattern is produced *by* the simulation from a
deterministic seed at run time. This one-line file lets `demo/run_demo` run
**offline, with zero downloads** (a hard requirement, CLAUDE.md §8).

### Field meanings (units are dimensionless; grid spacing Δx = 1)

| Field | Meaning | Sample |
|-------|---------|--------|
| `nx`, `ny` | grid width / height in cells | `64 64` |
| `Da` | activator diffusion coefficient (small) | `0.02` |
| `Dh` | inhibitor diffusion coefficient (large; `Dh ≫ Da` for a Turing instability) | `0.5` |
| `rho` | reaction strength (autocatalysis + inhibitor production) | `0.05` |
| `mu_a` | activator linear decay rate | `0.1` |
| `mu_h` | inhibitor linear decay rate | `0.14` |
| `rho_a` | basal (constant) activator source | `0` |
| `dt` | explicit-Euler timestep (must satisfy `dt·Dh·4 < 1`) | `0.4` |
| `steps` | number of timesteps to integrate | `3000` |
| `noise_seed` | seed for the deterministic initial perturbation | `12345` |

The committed default has `Dh/Da = 25`, squarely in the pattern-forming regime;
the demo's linear-stability check reports `Turing regime = YES`. Regenerate or
sweep with `python scripts/make_synthetic.py --Dh 0.2` etc.

## "Full dataset" — real-biology references (optional, for comparison)

There is no downloadable Turing dataset. `scripts/download_data.ps1` / `.sh`
**print instructions** (they download nothing automatically) pointing at the
public references named in the catalog, if you want to *visually* compare the
simulated pattern against real pigmentation or folding:

- **Pigmentation pattern images** (leopard, zebrafish) — public image sources;
  used only as qualitative visual comparisons, not as model input.
- **HCP cortical-folding atlases** — <https://db.humanconnectome.org> (requires
  registration and a data-use agreement; the script does **not** bypass it).
- **DANDI morphogenesis imaging** — <https://dandiarchive.org> (open neuroimaging
  archive).

Respect every license. None of these are redistributed here; the committed sample
is synthetic.

### Field meanings (units)

One liver **lobule** is modeled as `nsin` parallel **sinusoids** (capillaries).
Every sinusoid shares the physical constants; only its inlet blood velocity differs.

| Field | Meaning | Units |
|---|---|---|
| `L` | sinusoid length, periportal (`x=0`) → centrilobular (`x=L`) | mm |
| `C_in` | inlet drug/toxin concentration entering at `x=0` | µM |
| `Km` | Michaelis constant (concentration at half-maximal clearance) | µM |
| `Vmax_pp` | maximal clearance rate at the **periportal** end (zone 1) | µM/s |
| `Vmax_cl` | maximal clearance rate at the **centrilobular** end (zone 3) | µM/s |
| `nseg` | number of RK4 spatial steps along each sinusoid (grid resolution) | — |
| `nsin` | number of parallel sinusoids = ensemble members = GPU threads | — |
| `v_lo` | slowest inlet blood velocity in the perfusion sweep | mm/s |
| `v_hi` | fastest inlet blood velocity in the perfusion sweep | mm/s |

The sample values (`L=0.5, C_in=1, Km=50, Vmax_pp=8, Vmax_cl=2, nseg=200, nsin=4096,
v=0.2..1.0`) are physiologically **plausible** but invented. `C_in ≪ Km` keeps the
Michaelis-Menten reaction near its first-order regime, which lets `main.cu`
cross-check the numerics against a closed-form exponential washout (see THEORY §6).

## Full / real datasets

No public dataset is a drop-in for this loader — a calibrated model must be
*curated* from several sources. `scripts/download_data.{ps1,sh}` print the
provenance and, for credentialed sets, only the registration instructions (it
never bypasses a login):

- **Human Protein Atlas** — liver **zonal enzyme expression** — sets the `Vmax_pp`
  vs `Vmax_cl` gradient. https://www.proteinatlas.org (CC BY-SA 3.0).
- **HMDB** — liver **metabolite concentrations** — sets realistic `C_in`/`Km`
  scales. https://hmdb.ca (free for academic use; see terms).
- **Open Systems Pharmacology PBPK library** — organ-level **clearance parameters**.
  https://github.com/Open-Systems-Pharmacology/OSP-PBPK-Model-Library (GPLv2).
- **PhysioNet** — renal-function datasets (some **credentialed**).
  https://physionet.org (register; the script prints instructions only).

```
python scripts/make_synthetic.py --nsin 1048576  # a million sinusoids (stress the GPU)
```

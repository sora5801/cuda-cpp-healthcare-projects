This tiny file lets `demo/run_demo` run **offline, with zero downloads**, a hard
requirement for every project (CLAUDE.md §8). It is a single line of numbers.

### Layout (one whitespace-separated line, 16 numbers)

The C++ loader (`src/reference_cpu.cpp::load_ensemble`) reads bare numeric tokens
with `operator>>`, so the file contains **only numbers** (no comment lines):

```
alpha0 beta dt steps na nn alpha_lo alpha_hi n_lo n_hi  m0 m1 m2 p0 p1 p2
```

| Token | Meaning | Units |
|---|---|---|
| `alpha0` | leaky/basal transcription (promoter never fully off) | mRNA decay rate |
| `beta` | protein-decay / mRNA-decay rate ratio | dimensionless |
| `dt` | RK4 timestep | mRNA lifetimes |
| `steps` | number of timesteps (run length = `steps*dt`) | count |
| `na`, `nn` | sweep grid: `na` α-values × `nn` Hill-coefficient values | count |
| `alpha_lo..hi` | max transcription-rate sweep range | mRNA decay rate |
| `n_lo..hi` | Hill-coefficient (cooperativity) sweep range | dimensionless |
| `m0 m1 m2` | initial mRNA concentrations (shared IC) | dimensionless |
| `p0 p1 p2` | initial protein concentrations (shared IC) | dimensionless |

The committed sample is a **6×6 = 36-member** sweep of `alpha ∈ [10,260]` and
`n ∈ [1,3]`, with an asymmetric seed `m0=1` (all else 0) so the ring is not stuck
at its symmetric fixed point. Low-`n` members settle to a steady state; high-`n`,
high-`alpha` members enter the famous sustained **oscillation** — that emergent
genetic clock is the "known answer" the demo recovers (PATTERNS.md §6).

Regenerate or resize it with:

```
python scripts/make_synthetic.py                  # default 6x6
python scripts/make_synthetic.py --na 64 --nn 64  # 4096-member sweep
```

## Full dataset (real models)

Real systems-biology models are published as **SBML** (XML) files in curated
repositories. There is no single tabular download; `scripts/download_data.*`
prints these pointers (no credentials are required for any of them):

- **BioModels Database (EMBL-EBI)** — 1000+ curated SBML models,
  <https://www.ebi.ac.uk/biomodels>. The repressilator itself is `BIOMD0000000012`.
- **Reactome pathways** — curated molecular interactions, <https://reactome.org>.
- **BioGRID** — interaction network, <https://thebiogrid.org>.
- **VCell** — curated cell models, <https://vcell.org>.

**Why we do not ship a real SBML file:** turning an arbitrary SBML model into an
ODE right-hand side is a parsing + code-generation task (that is exactly what
libRoadRunner / Tellurium do — see README "Prior art & further reading"). That is
deliberately **out of scope** for this teaching demo, which hard-codes the
repressilator RHS in `src/grn.h` so the lesson stays on the **GPU batched-ODE
pattern**. Respect each repository's license before redistributing anything.

### Why synthetic, and what the ground truth is

Real single-cell RNA-seq (GEO, Human Cell Atlas) is large, license-bound, and —
crucially for a teaching demo — comes with **no known regulatory network** to
check the answer against. So the sample is deliberately synthetic with a
**planted ground-truth graph** the algorithm should recover:

```
        TF ──► A ──► B        (a linear cascade: TF regulates A, A regulates B)
        TF ──► C              (a direct target of TF)
        D  ──► E              (a second, TF-independent regulatory pair)
        F, G, H, I            (unconnected NOISE genes)
```

Each gene's expression in a given cell is a **noisy linear function of its
regulator(s)** (Gaussian transcriptional noise), then quantized to a few discrete
levels — mimicking how discrete, noisy real counts look. Because `B` is driven by
`A` which is driven by `TF`, raw pairwise mutual information sees a *spurious*
`TF–B` correlation; ARACNE's Data Processing Inequality removes it, recovering the
true chain. That prune is the point of the demo. **Everything here is synthetic —
no clinical meaning whatsoever.**

Regenerate (byte-identical, fixed seed) or scale up with:

```
python scripts/make_synthetic.py                 # the committed 10×200 sample
python scripts/make_synthetic.py --samples 400   # more cells (sharper MI)
```

## Full / real dataset

There is no single file to download; you supply an expression matrix
(genes × cells) and reshape it to the loader format above. Sources (from the
catalog `Datasets` column) — respect each license; none are redistributed here:

- **Gene Expression Omnibus (GEO)** — tens of thousands of scRNA-seq datasets:
  <https://www.ncbi.nlm.nih.gov/geo/>
- **BEELINE benchmark GRN datasets** (curated, with reference networks):
  <https://github.com/Murali-group/BEELINE>
- **Human Cell Atlas scRNA-seq**: <https://www.humancellatlas.org>
- **ENCODE TF-binding ChIP-seq** (a source of *ground-truth* TF→gene edges):
  <https://www.encodeproject.org>

`scripts/download_data.ps1` / `.sh` print these links and the reshape recipe; they
download nothing by themselves and never bypass registration/credentials.

## Provenance & field meanings

- **`<name>`** — a gene label (string). In the sample: `TF, A, B, C, D, E, F, G, H, I`.
- **`v0 … v(S-1)`** — that gene's expression across the `S` samples (cells), one
  number per cell. In the sample these are small non-negative integers (quantized
  synthetic levels); real data can be raw counts or normalized values (the loader
  reads any real numbers, and each gene is discretized against its own range).

Never imply clinical validity; synthetic data is labeled synthetic everywhere.

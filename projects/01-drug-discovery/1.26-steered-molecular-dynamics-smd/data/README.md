|---|---|

| Origin | **Synthetic** (generated by `scripts/make_synthetic.py`) |

| Layout | one whitespace-separated line (see field table below) |

This tiny file lets `demo/run_demo` run **offline, with zero downloads**, a hard
requirement for every project (CLAUDE.md §8). It is **not** a molecular structure
— it parameterises a reduced 1-D SMD model whose free-energy answer is known
analytically, so the demo can check itself (see `../THEORY.md`).

### Field meanings (in file order)

```
xi0 xi_end n_traj steps dt k_spring v_pull gamma kT pmf_A pmf_xa pmf_xb pmf_slope seed
```

| Field | Meaning | Units | Sample |
|---|---|---|---|
| `xi0` | bound-state reaction coordinate (pull start) | nm | 0 |
| `xi_end` | unbound-state reaction coordinate (pull end) | nm | 1 |
| `n_traj` | number of independent SMD trajectories (the ensemble) | — | 8192 |
| `steps` | Langevin integrator steps per trajectory | — | 25000 |
| `dt` | timestep | ps | 0.002 |
| `k_spring` | SMD harmonic-spring stiffness | kJ/mol/nm² | 2000 |
| `v_pull` | dummy-atom pulling velocity | nm/ps | 0.02 |
| `gamma` | Langevin friction coefficient | (kJ/mol)·ps/nm² | 500 |
| `kT` | thermal energy kB·T (T = 300 K) | kJ/mol | 2.4943 |
| `pmf_A` | PMF (potential of mean force) barrier scale | kJ/mol | 25 |
| `pmf_xa` | bound-well centre | nm | 0 |
| `pmf_xb` | unbound-well centre | nm | 1 |
| `pmf_slope` | linear tilt of the PMF; **sets the true ΔG** | kJ/mol/nm | -12 |
| `seed` | base RNG seed (per-trajectory streams derive from it) | — | 20240626 |

The geometry is self-consistent: `v_pull · steps · dt = 0.02 · 25000 · 0.002 =
1.0 nm = xi_end − xi0`, so the dummy atom travels exactly from the bound to the
unbound coordinate. The **true** end-to-end free energy is
`ΔG = pmf_slope · (xi_end − xi0) = −12 kJ/mol` (the double-well quartic vanishes
at both wells), which is the ground truth the Jarzynski estimate is checked
against.

Regenerate or rescale it:

```
python scripts/make_synthetic.py                 # the committed default
python scripts/make_synthetic.py --n-traj 65536  # tighter Jarzynski convergence
```

## Full dataset

There is **no file to download** for this teaching project — the model is fully
specified by the 14 numbers above. Real SMD operates on full-atom molecular
systems and needs a production MD engine. Pointers (catalog "Datasets" /
"Starter repos"):

- **NAMD SMD tutorials** — <https://www.ks.uiuc.edu/Training/Tutorials/> (the
  canonical constant-velocity / constant-force SMD walkthroughs, e.g. pulling
  deca-alanine, with VMD setup).
- **BindingDB residence-time data** — <https://www.bindingdb.org> (kinetics /
  k_off data that SMD rupture-force studies relate to).
- **PDB** — <https://www.rcsb.org> (structures used as force-probe starting
  points in published SMD studies of ion channels and motor proteins).

Real SMD inputs are subject to the licenses of each structure/dataset; respect
them. `scripts/download_data.ps1` / `.sh` print these pointers and never attempt
to bypass any registration.

## Honesty / safety

The committed data is **synthetic and labeled synthetic everywhere**. The 1-D
model is a didactic caricature of full-atom SMD; its numbers are pedagogical, not
a property of any real molecule, and nothing here is for clinical or design use
(CLAUDE.md §1, §8).

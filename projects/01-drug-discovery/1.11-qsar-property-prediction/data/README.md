### File format (parsed by `src/reference_cpu.cpp::load_graph`)

A batched-graph **CSR** layout — the same idea PyTorch Geometric / DGL use to
pack many small graphs into flat arrays:

```
line 1            : num_mols  num_nodes  num_edges     (num_edges = REAL bonds;
                    the loader adds a self-loop to every atom, not listed here)
next num_nodes    : 6 feature floats per atom (one atom per line)
next num_mols     : atom_count for each molecule (must sum to num_nodes)
next num_edges    : "u v" undirected bond, GLOBAL atom indices (0-based)
```

Atom indices are **global across the batch**: molecule 0's atoms come first,
then molecule 1's, and so on (`mol_start` in `Graph` records the boundaries).

### The 6 atom features (a tiny, didactic descriptor vector)

| Index | Meaning |
|---|---|
| `f0` | 1 if the atom is **carbon-like** (one-hot element bit) |
| `f1` | 1 if the atom is **nitrogen-like** |
| `f2` | 1 if the atom is **oxygen-like** |
| `f3` | normalized local degree (`#bonds / 4`) — a cheap topological descriptor |
| `f4` | 1 if the atom is **in a ring** (synthetic ring flag) |
| `f5` | constant `1.0` **bias channel** (lets the network learn an offset) |

The five toy molecules are a butane-like chain, a 6-membered carbon ring, an
ethanolamine-like N–C–C–O chain, a branched C(C,N,O) star, and a 5-membered ring
with one nitrogen. Real featurizers (RDKit) emit dozens of bits — hybridization,
formal charge, aromaticity, H-count, chirality — here six keep the file readable.

This tiny file lets `demo/run_demo` run **offline, with zero downloads**, a hard
requirement for every project (CLAUDE.md §8). Regenerate it with:

```
python scripts/make_synthetic.py
```

## Full / real datasets

Real QSAR benchmarks ship as **SMILES strings + measured labels**. To use them
you featurize each SMILES into an atom/bond graph (RDKit) and emit the CSR format
above. `scripts/download_data.{ps1,sh}` print the links and the conversion recipe;
they never bypass credentials.

| Dataset | What it is | URL | Access |
|---|---|---|---|
| **MoleculeNet** | 17+ curated property/activity benchmarks (ESOL, FreeSolv, Lipophilicity, BBBP, Tox21, …) | https://moleculenet.org | open, no login |
| **ChEMBL** | measured bioactivities for ~2.4M compounds | https://www.ebi.ac.uk/chembl/ | open bulk download |
| **TDC** (Therapeutics Data Commons) | 66 ready-made drug-discovery ML tasks | https://tdcommons.ai | `pip install PyTDC` |
| **PCBA** | 128 PubChem BioAssays over ~440k compounds | https://moleculenet.org | open |

**License:** each source has its own terms — respect them. Because these labels
are measured bioactivity we do **not** redistribute any of them here; the
committed sample is fully synthetic.

## Honesty / safety

- The weights are **untrained** (seeded LCG), so the printed numbers carry **no
  predictive meaning** — they exist only to demonstrate the GPU pipeline and the
  CPU↔GPU agreement.
- Synthetic data is labeled synthetic everywhere it appears.
- Nothing here is for diagnosis, treatment, or real molecular design.

### Field meanings (units in a reduced system where the Boltzmann constant kB = 1)

| Field | Meaning | Sample value |
|---|---|---|
| `kA`      | state-A spring constant (energy / length²) | `1.0` |
| `x0A`     | state-A equilibrium position (length)      | `0.0` |
| `kB`      | state-B spring constant (energy / length²) | `4.0` |
| `x0B`     | state-B equilibrium position (length)      | `1.0` |
| `kT`      | temperature in energy units (`kB·T`)       | `1.0` |
| `windows` | number of λ-windows incl. endpoints (≥ 2)  | `11`  |
| `equil`   | MC burn-in steps discarded per window      | `2000` |
| `samples` | MC steps averaged per window               | `20000` |
| `step`    | MC trial-move half-width (length)          | `0.6` |
| `x_init`  | starting position of every MC chain        | `0.0` |

**Why these values are interpretable.** With `kA=1, kB=4, kT=1`, the closed-form
free-energy difference is `ΔG = ½·kT·ln(kB/kA) = ½·ln(4) = ln(2) ≈ 0.693147`. The
demo's Thermodynamic-Integration estimate should land near this number — that is
how you can tell the calculation is working (a known answer embedded in the data,
per `docs/PATTERNS.md §6`).

Regenerate or vary it:

```bash
python scripts/make_synthetic.py                 # the default sample above
python scripts/make_synthetic.py --kB 9          # ΔG = ½·ln(9) ≈ 1.0986
python scripts/make_synthetic.py --windows 21    # finer λ-grid (smaller TI bias)
```

## Full / real datasets

This is a **reduced-scope teaching model**, so there is no real dataset to load —
a faithful FEP/TI calculation needs a full MD engine and a molecular force field
(see `THEORY.md` "Where this sits in the real world"). For study, the catalog
points at these real benchmarks (handled by `scripts/download_data.*`, which only
prints links/instructions and never bypasses any registration):

> Merck FEP benchmark set — 8 targets with experimental ΔΔG (available via OpenFE;
> https://github.com/OpenFreeEnergy/openfe); FEP+ validation set (Schrödinger,
> verify URL); PDBbind — experimental binding affinities (http://www.pdbbind.org.cn);
> ChEMBL activity data for target families (https://www.ebi.ac.uk/chembl/).

- **License:** respect each source's terms. The Merck/OpenFE set is open; FEP+ and
  some PDBbind tiers require registration. Because these cannot be repackaged here,
  the committed sample is **synthetic** and `make_synthetic.py` is the stand-in.
- **Not for clinical use.** Nothing here is a validated affinity prediction; the
  numbers are educational and the data is synthetic wherever it appears.

# Data — 1.7 Quantum Chemistry / DFT

## Committed sample (`sample/`)

| Field | Value |
|---|---|
| File | `sample/h2.txt` |
| Origin | **Synthetic / public** — a hand-written textbook geometry, generated by `scripts/make_synthetic.py` |
| License | Public domain (CC0) — it is a standard physical constant, not data |
| Size | < 1 KB |
| Contents | The hydrogen molecule (H₂) at its equilibrium bond length |

This tiny file lets `demo/run_demo` run **offline, with zero downloads**, which is
a hard requirement for every project (CLAUDE.md §8). It is **not** patient-derived
and carries no clinical meaning — it is the geometry of two protons.

### File format

```
# comment lines start with '#'
<natoms> <charge>
<Z> <x> <y> <z>        # one line per atom
...
```

| Field | Meaning | Units |
|---|---|---|
| `natoms` | number of atoms | integer |
| `charge` | net molecular charge (0 = neutral, 1 = cation +1) | integer |
| `Z` | nuclear charge / atomic number (H = 1, He = 2) | integer |
| `x y z` | nucleus position | **Bohr** (atomic units; 1 Bohr ≈ 0.529 Å) |

The number of electrons is computed as `sum(Z) − charge` and must be **even**
(this teaching build does closed-shell restricted Hartree–Fock only).

The committed `sample/h2.txt` is H₂ at R = 1.4 Bohr. With the STO-3G basis this
reproduces the canonical textbook total energy **−1.1167 Hartree** (Szabo &
Ostlund, *Modern Quantum Chemistry*).

### Generate more inputs

```bash
python scripts/make_synthetic.py --mol h2     # the committed H2 (default)
python scripts/make_synthetic.py --mol heh+   # HeH+ cation (2 electrons)
python scripts/make_synthetic.py --mol he     # a single He atom
```

Only H and He are supported because the basis set in this teaching build ships
only their STO-3G parameters (extending it is an exercise — see the README).

## Full datasets (reference only — not needed by the demo)

The catalog lists large research corpora of **precomputed** quantum-chemistry
results. They are used to train/benchmark machine-learning models, **not** as
inputs to this SCF program, so the demo does not download them. `scripts/download_data.*`
prints these pointers and fetches nothing.

Catalog dataset notes (verbatim):

> QM9 — DFT-computed properties of 134k organic molecules (https://doi.org/10.6084/m9.figshare.978904); ANI-1ccx — CCSD(T)-level energies for diverse organic molecules (https://github.com/isayev/ANI1ccx_dataset); PubChemQC — DFT calculations for ~3M PubChem molecules (http://pubchemqc.riken.jp); CSD — Cambridge Structural Database for crystal structures (https://www.ccdc.cam.ac.uk).

- **Licenses:** QM9 and PubChemQC are open for research; the **CSD requires a
  license** — the download script never bypasses that, it only links to it.
- **Provenance:** all are computed/curated chemistry data, none patient-derived.

## Safety note

Everything here is **educational**. The energies this project computes are real
quantum chemistry but on toy two-electron systems; nothing here is a clinical or

## Committed sample (`sample/`)

| Field | Value |
|---|---|
| File | `sample/metad_config.txt` |
| Origin | **Synthetic** (a run configuration, not measured data; generated by `scripts/make_synthetic.py`) |
| License | Public domain (CC0) — it is synthetic |
| Size | < 1 KB (one line of numbers) |
| Layout | a single whitespace-separated line (16 fields; see below) |

This tiny file lets `demo/run_demo` run **offline, with zero downloads**, which is
a hard requirement for every project (CLAUDE.md §8). It is **not** a molecular
trajectory — it is the *configuration* of a synthetic 1-D metadynamics run. The
landscape it describes (a double well with a known analytic free energy) is chosen
precisely so the recovered free-energy surface can be **verified** against the
exact answer.

### Field meanings (single line, in order)

```
A kT mass friction dt steps hill_w hill_sigma deposit_every bias_factor
s_lo s_hi nbins n_walkers seed s_start
```

| Field | Meaning | Units | Sample |
|---|---|---|---|
| `A` | barrier height of the double well `F0(s)=A(s²−1)²` | kT | 5 |
| `kT` | thermal energy `k_B·T` (noise strength + temper scale) | energy | 1 |
| `mass` | effective CV mass | reduced | 1 |
| `friction` | Langevin friction `γ_L` (damping) | 1/time | 2 |
| `dt` | integrator timestep | time | 0.005 |
| `steps` | Langevin steps per walker | — | 20000 |
| `hill_w` | Gaussian hill height before tempering | energy | 0.3 |
| `hill_sigma` | Gaussian hill width along `s` | CV units | 0.1 |
| `deposit_every` | metadynamics pace (deposit one hill per N steps) | steps | 50 |
| `bias_factor` | well-tempered bias factor `γ` (>1) | — | 10 |
| `s_lo`, `s_hi` | bias-grid extent along the CV | CV units | −2, 2 |
| `nbins` | bias-grid resolution (FES bins) | — | 121 |
| `n_walkers` | independent walkers (= GPU threads) | — | 64 |
| `seed` | base RNG seed (walker id mixes in per stream) | — | 20240617 |
| `s_start` | \|start\| CV magnitude; walkers alternate ± | CV units | 1 |

Everything else (each walker's RNG stream and start well) is **derived
deterministically**, so the whole run is reproducible from this one line.

## Full / real datasets

There is **no real dataset to download** for this teaching project — the input is
a synthetic configuration. For *real* enhanced-sampling study material, the
catalog points to published inputs and trajectories. `scripts/download_data.ps1`
(and `.sh`) print links and instructions; they never bypass any registration.

Catalog dataset notes (verbatim):

> PLUMED-NEST — repository of published metadynamics/enhanced sampling input files
> (https://www.plumed-nest.org); GPCRmd trajectory archive (https://gpcrmd.org);
> D. E. Shaw millisecond MD datasets (available via RCSB); benchmark FES for
> alanine dipeptide / chignolin (commonly used test systems).

- **PLUMED-NEST** — open repository of reproducible PLUMED inputs (metadynamics,
  REST2, funnel MetaD…). Free, no registration. The natural next step after this
  toy model: run a real well-tempered MetaD on **alanine dipeptide**.
- **GPCRmd** — curated GPCR MD trajectories (free account for bulk download).
- **D. E. Shaw long-timescale MD** — available through RCSB / by request.

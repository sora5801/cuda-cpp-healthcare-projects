| Field | Value |

| Why synthetic | Surrogate scores are engineered to span a wide range and to embed a **known answer** (4 designed binders), so the top-K hit list is interesting **and deterministic**. |

This tiny file lets `demo/run_demo` run **offline, with zero downloads**, which is
a hard requirement for every project (CLAUDE.md §8).

### File format

```
# comment lines (start with '#') and blank lines are ignored
n                                                  # number of ligands (e.g. 64)
TARGET mw_opt logp_opt_x100 psa_opt feat_required  # one target line
mw logp_x100 hbd hba rotb psa feat_hex             # one line per ligand (n lines)
```

Each **ligand** is a small 2-D descriptor vector plus a 32-bit pharmacophore
feature bitmask (written as hex, e.g. `0x00A5B3`):

| Column | Meaning | Units / range |
|---|---|---|
| `mw` | molecular weight | Daltons (integer-rounded) |
| `logp_x100` | lipophilicity (logP) ×100, signed | e.g. `250` = logP 2.50 |
| `hbd` | hydrogen-bond donors | count |
| `hba` | hydrogen-bond acceptors | count |
| `rotb` | rotatable bonds (flexibility) | count |
| `psa` | topological polar surface area | Å² |
| `feat_hex` | pharmacophore feature bitmask | 32-bit hex |

The **TARGET** line gives the pocket's ideal `mw_opt / logp_opt_x100 / psa_opt`
and the `feat_required` bitmask (features the pocket rewards). The loader
(`src/reference_cpu.cpp::load_library`) rejects a malformed file loudly.

The committed sample contains, after a deterministic shuffle: **4 designed
binders** (every rewarded feature, on-target properties → they top the ranking),
**8 filter-fails** (deliberately violating a Lipinski/Veber rule → rejected by
the cascade), and **52 decoys** (random but drug-like → middling scores).

## Full dataset

Real screening descriptors are computed from molecule libraries with **RDKit**
(SMILES → MW/logP/HBD/HBA/rotatable bonds/PSA + a Morgan/pharmacophore feature
bitmask). `scripts/download_data.ps1` / `.sh` print the recipe and the links:

- **Enamine REAL** — >6B make-on-demand compounds: <https://enamine.net/compound-collections/real-compounds>
- **ZINC20** — free virtual-screening DB, ~2B purchasable: <https://zinc20.docking.org>
- **ChEMBL** — bioactivity reference: <https://www.ebi.ac.uk/chembl/>
- **ExCAPE-DB** — large-scale public chemogenomics: <https://solr.ideaconsult.net/search/excape/>

For a large **synthetic** library instead (no download, fully offline):

```
python scripts/make_synthetic.py --n 1000000
```

## Provenance & honesty

The committed sample is **synthetic** and labelled as such everywhere. The
descriptor values and scores are **chemically meaningless** — they exist only to
exercise and verify the kernel and to plant a recoverable top-K. No clinical or
chemical conclusion may be drawn from this sample.

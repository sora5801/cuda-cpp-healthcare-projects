# Data — 1.12 Molecular Fingerprint Similarity Search

## Committed sample (`sample/fingerprints_sample.txt`)

| Field | Value |
|---|---|
| Origin | **Synthetic** (generated by `scripts/make_synthetic.py`, seed 7) |
| License | Public domain (CC0) — it is synthetic |
| Size | 64 library fingerprints + 1 query, **2048-bit** (32×64) each (~9 KB) |
| Why synthetic | Tanimoto scores are engineered to span a wide range so the top-K hit list is interesting and **deterministic**. |

### File format

```
<n> <FP_WORDS>                         # e.g. "64 32"  (FP_WORDS must be 32)
<query: FP_WORDS hex words>            # 32 space-separated 16-hex-digit 64-bit words
<library fp 0: FP_WORDS hex words>
<library fp 1: ...>
... (n lines total)
```

Each fingerprint is a 2048-bit vector packed into 32 little-endian 64-bit words,
written as 16-hex-digit tokens. Bit *k* present ⇒ chemical substructure *k*
present. The loader (`src/reference_cpu.cpp::load_fingerprints`) rejects a file
whose `FP_WORDS` differs from the compiled constant.

## Full dataset

Real fingerprints are generated from molecule libraries with **RDKit**:

- **ChEMBL** — bioactivity DB, >2M compounds: <https://www.ebi.ac.uk/chembl/>
- **ZINC20** — purchasable compounds: <https://zinc20.docking.org>
- **PubChem** — 115M+ compounds: <https://pubchem.ncbi.nlm.nih.gov>
- **Enamine REAL** — multi-billion make-on-demand: <https://enamine.net>

`scripts/download_data.ps1` / `.sh` print the RDKit recipe (SMILES → Morgan/ECFP4
→ this hex format). For a large synthetic library instead:

```
python scripts/make_synthetic.py --n 1000000
```

## Provenance & honesty

The committed sample is **synthetic** and labeled as such everywhere. Scores are
chemically meaningless — they exist only to exercise and verify the kernel. No
clinical or chemical conclusion may be drawn from this sample.

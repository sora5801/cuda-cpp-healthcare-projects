### File format (whitespace-separated text)

```
line 1 :  ncell nord width max_iter tol psi_left_bc psi_right_bc
lines 2..ncell+1 :  sigma_t sigma_s q      (one physical cell per line)
```

| Symbol | Meaning | Units |
|---|---|---|
| `ncell` | number of spatial cells along x | — |
| `nord` | S_N order = number of discrete ordinates (even, ≥2) | — |
| `width` | total slab thickness | cm |
| `max_iter` | source-iteration cap (safety) | — |
| `tol` | convergence tolerance (relative L∞ on the scalar flux) | — |
| `psi_left_bc`, `psi_right_bc` | incoming angular flux at the two faces (0 = vacuum) | flux |
| `sigma_t` | total macroscopic cross-section of the cell | 1/cm |
| `sigma_s` | scattering macroscopic cross-section (≤ `sigma_t`) | 1/cm |
| `q` | fixed isotropic external source in the cell | particles/cm³/s |

Implied per cell: absorption `sigma_a = sigma_t − sigma_s` (drives the dose
proxy). The committed slab uses `sigma_t=1.0, sigma_s=0.8` in the outer "tissue"
thirds (scatter ratio 0.8) and `sigma_t=0.2, sigma_s=0.15` in the low-density
"lung" middle third, with a unit source in a thin band in the left tissue layer.
This tissue↔lung contrast is exactly where deterministic transport historically
disagreed with pencil-beam superposition — the reason Acuros XB exists.

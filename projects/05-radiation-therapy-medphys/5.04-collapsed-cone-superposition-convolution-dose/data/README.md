### File layout

The file is whitespace-separated plain text:

```
line 1 : nx ny voxel_cm mu_over_rho psi0 n_cones kernel_a dose_scale beam_x0 beam_x1
then   : nx*ny density values, row-major (y = 0 is the TOP / entry row)
```

### Field meanings (units)

| Field | Meaning | Units |
|---|---|---|
| `nx`, `ny` | grid width, height | voxels |
| `voxel_cm` | physical edge length of one square voxel | cm |
| `mu_over_rho` | mass attenuation coefficient μ/ρ of the medium | cm²/g |
| `psi0` | incident primary photon fluence at the top surface | arbitrary |
| `n_cones` | number of collapsed-cone directions (1–8 in this model) | count |
| `kernel_a` | cone dose-spread kernel decay constant *a* | 1/(g·cm⁻²) |
| `dose_scale` | fixed-point quantum: `dose_units = round(dose × dose_scale)` | units/dose |
| `beam_x0`, `beam_x1` | first, last irradiated column (inclusive) | voxel index |
| density values | ρ per voxel, **water-relative** (water = 1.0) | dimensionless |

### What the phantom contains

A horizontal-slab stack the beam enters from the top (`y = 0`), designed so the
central-axis depth-dose curve visibly shows **heterogeneity correction**:

| Rows | Tissue | ρ (water-relative) | Effect on dose spread |
|---|---|---|---|
| 0–3   | water | 1.00 | build-up region |
| 4–7   | lung  | 0.25 | low density → kernel reaches **farther** |
| 8–11  | water | 1.00 | soft tissue again |
| 12–13 | bone  | 1.85 | high density → kernel reaches **less**, dose piles up |
| 14–15 | water | 1.00 | |

## Full / reference datasets

The real dose-engine benchmarks named in the catalog are for going further. They
require registration or have redistribution limits, so we do **not** commit them
and `scripts/download_data.*` only prints links and instructions (never bypasses
credentials — CLAUDE.md §8):

- **AAPM TG-105** test cases (heterogeneous-media dose benchmarks) —
  <https://www.aapm.org/pubs/reports/> (search "TG-105"). The canonical reference
  for benchmarking convolution/superposition and Monte-Carlo dose engines.
- **IROC Houston** lung-phantom CT + dosimetry (clinical-trial credentialing).
- **TCIA** clinical photon planning datasets (CT + RTDOSE/RTPLAN DICOM) —
  <https://www.cancerimagingarchive.net/> (register, then browse the RT collections).
- **CIRS** IMRT verification phantom data — <https://www.cirsinc.com/>.

Respect every license. If a set forbids redistribution, keep the committed sample
synthetic (as here).

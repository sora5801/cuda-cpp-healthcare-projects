### File format (parsed by `src/reference_cpu.cpp::load_stray_problem`)

Lines beginning with `#` are comments. The first data line holds 11 numbers:

| Field | Meaning | Units |
|---|---|---|
| `field_end` | organ index where the treatment field ends (organs `0..field_end-1` are in-field) | index |
| `mu` | linear attenuation coefficient of tissue | 1/cm |
| `organ_cm` | thickness of each organ slab along the body axis | cm |
| `scatter_frac` | fraction of an interacting photon's weight that scatters (survives) vs. is absorbed | — |
| `sidescatter` | forced-detection lateral-scatter fraction redirected to each downstream organ | — |
| `leakage_frac` | machine-head leakage weight added uniformly to every organ per primary | — |
| `neutron_frac` | secondary-neutron **surrogate** weight (distance-weighted); see THEORY.md | — |
| `roulette_floor` | statistical weight below which Russian roulette is played | — |
| `roulette_survive` | survival probability in roulette (weight boosted by `1/survive`) | — |
| `n_histories` | number of primary photon histories to simulate | count |
| `seed` | base RNG seed (history `i` uses stream `(seed, i)`) | integer |

Each subsequent data line is one organ: `<name> <risk_coeff>`. `n_organs` is the
number of organ lines. Organ `0` is the treated **Target**; the rest are
out-of-field organs whose dose is entirely stray. `risk_coeff` is an illustrative
BEIR-VII-style relative cancer-risk sensitivity (**not clinical**).

## Full / real datasets (pointers only — not committed)

The catalog names these real inputs. The download script prints instructions and
never bypasses any registration (CLAUDE.md §8):

- **ICRP 110 voxel phantoms** (adult male/female) —
  <https://www.icrp.org/publication.asp?id=ICRP%20Publication%20110>. The real
  geometry: 3-D labelled organ voxels replacing our 1-D organ stack.
- **NIST XCOM photon cross-sections** —
  <https://www.nist.gov/pml/xcom-photon-cross-sections>. Energy-dependent
  attenuation/scatter coefficients replacing our single constant `mu`.
- **Secondary-dose measurements from the literature** — for validating out-of-field
  dose falloff.
- **TCIA proton-therapy planning CTs** — <https://www.cancerimagingarchive.net/>
  (registration/attribution required) for realistic patient geometry.

Fetch guidance: `scripts/download_data.ps1` (Windows) / `scripts/download_data.sh`
(Linux/macOS).

## Safety

Everything here is **synthetic and educational**. No output may be used for
diagnosis, treatment, or any real medical decision. The risk coefficients are
illustrative teaching values, not BEIR-VII clinical coefficients.

## Committed sample (`sample/flash_ensemble.txt`)

| Field | Value |
|---|---|
| File | `sample/flash_ensemble.txt` |
| Origin | **Synthetic** (generated by `scripts/make_synthetic.py`) |
| License | Public domain (CC0) — it is synthetic |
| Size | < 1 KB (a single line of 9 numbers) |

The sample is **not** a bulk dataset — it is the small **ensemble configuration**
that drives the per-voxel radiation-chemistry simulation. All the physics is in
the code (`src/flash.h`); this file only says *which sweep to run*.

### Layout (one whitespace-separated line)

```
total_dose  n_pulses  dt  conv_steps_per_gap  flash_steps_per_gap  relax_steps  n_po2  po2_lo  po2_hi
```

| Field | Meaning | Units | Sample value |
|---|---|---|---|
| `total_dose` | total prescribed dose (split evenly over the pulses) | Gy | 10 |
| `n_pulses` | number of beam pulses in the train | — | 10 |
| `dt` | RK4 integration timestep | s | 1e-05 |
| `conv_steps_per_gap` | RK4 sub-steps between pulses for **conventional** delivery (long gap → O₂ refills) | — | 4000 |
| `flash_steps_per_gap` | RK4 sub-steps between pulses for **FLASH/UHDR** delivery (tiny gap → O₂ cannot refill) | — | 1 |
| `relax_steps` | post-delivery relaxation sub-steps (both modes) | — | 4000 |
| `n_po2` | number of oxygen-tension levels swept | — | 8 |
| `po2_lo`, `po2_hi` | pO₂ sweep range | mmHg | 2 … 40 |

The ensemble is `n_po2 × 2 delivery modes = 16` independent per-voxel solves.
The conventional inter-pulse gap is `conv_steps_per_gap · dt = 0.04 s` (40 ms);
the FLASH gap is `flash_steps_per_gap · dt = 1e-5 s` (10 µs). Because the
vasculature's O₂ recovery time-constant is ~3 ms (see `src/flash.h`
`default_rates`), O₂ **fully refills** during a conventional gap but **cannot**
during a FLASH gap — which is the entire modelled mechanism.

This tiny file lets `demo/run_demo` run **offline, with zero downloads**
(CLAUDE.md §8). Regenerate or resize it with:

```bash
python scripts/make_synthetic.py                 # defaults (the committed sample)
python scripts/make_synthetic.py --n-po2 32      # finer oxygen sweep
```

## Full / real-world datasets

There is **no bulk dataset to download** for this teaching project. For readers
who want to extend the model toward validation, the relevant real-world data
(all credentialed or not redistributable — respect each license) is:

- **FLASH-RT experimental dosimetry** — CERN/CLEAR, UCLouvain, Stanford FLASH
  programs (institutional access; verify each program's data-sharing policy).
- **AAPM FLASH-RT working-group benchmark datasets** (verify the current URL).
- **Tumour oxygen-tension (pO₂) measurements** — published Eppendorf-electrode
  and EPR-oximetry studies establish the hypoxic (<10 mmHg) vs normal-tissue
  (20–40 mmHg) ranges used here.
- **Geant4-DNA radiolysis validation datasets** — <https://geant4-dna.org> — the
  gold-standard track-structure water-radiolysis yields the lumped `g_rad` here
  abstracts away.

`scripts/download_data.ps1` / `.sh` print these pointers and never attempt to
bypass any registration.

## Provenance & honesty

The rate constants and yields in `src/flash.h` are **illustrative lumped
values**, chosen so the demo is interpretable — they are **not fitted** to any
measurement. The absolute damage numbers are in arbitrary "Gy-equivalent" units;
only the **conventional-vs-FLASH ratio** (the sparing factor) carries the
teaching message, and even that is qualitative. See `THEORY.md` for what a
research-grade model (track-structure Monte Carlo + full radiolysis networks)
does differently.

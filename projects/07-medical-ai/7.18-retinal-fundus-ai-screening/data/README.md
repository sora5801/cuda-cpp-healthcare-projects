# Data — 7.18 Retinal Fundus AI Screening

## Committed sample (`sample/`)

| Field | Value |
|---|---|
| File | `sample/fundus_sample.txt` |
| Origin | **Synthetic** (generated by `scripts/make_synthetic.py`) |
| License | Public domain (CC0) — it is synthetic, not a real retina |
| Size | ~21 KB |
| Contents | one 3×32×32 colour "fundus-like" image, pixels normalized to `[0,1]` |

This tiny file lets `demo/run_demo` run **offline, with zero downloads**, which
is a hard requirement for every project (CLAUDE.md §8). It is a hand-drawn
approximation of a fundus photograph — a warm orange-red disc with a bright
optic-disc-like blob, a curved vessel-like arc, and a few dark-red
microaneurysm-like spots — engineered so the CNN's fixed edge/blob/colour
filters have something to fire on and the Grad-CAM heatmap is interpretable
(PATTERNS.md §6). **It is not a real retina and must never be presented as one.**

### File layout (matches `load_fundus()` in `src/reference_cpu.cpp`)

```
line 1 :  C H W label        # C=3 (RGB), H, W, label = ground-truth DR grade or -1
then   :  C*H*W floats in [0,1]
```

The floats are **channel-major** then row-major: all of the R plane (row by
row), then the entire G plane, then the entire B plane. The line breaks are
cosmetic (16 values per line); the loader reads whitespace-separated tokens.

Regenerate or resize it:

```bash
python scripts/make_synthetic.py --size 32     # default committed sample
python scripts/make_synthetic.py --size 64     # a larger synthetic image
```

## Full dataset

The real, research-grade datasets for diabetic-retinopathy screening are large
and access-restricted. `scripts/download_data.*` print instructions and links;
they never bypass any login or license.

- **EyePACS** — ~88,000 labelled colour fundus images, 5-grade DR severity.
  Kaggle "Diabetic Retinopathy Detection" competition (account + rules
  acceptance required). Verify the current URL on Kaggle.
- **APTOS 2019** — 3,662 fundus images, DR grading. Kaggle "APTOS 2019 Blindness
  Detection" (account required). Verify URL.
- **DRIVE / STARE** — retinal vessel-segmentation datasets (smaller; registration
  varies). Verify URL.
- **UK Biobank Retinal Imaging** — ~68k fundus images with linked health records,
  strictly credentialed: <https://www.ukbiobank.ac.uk/>.

### Using real data with this teaching model

This project's CNN expects a small fixed-size RGB tensor in `[0,1]`. To try a
real image you would: load it (e.g. with Pillow), resize to a small square,
divide by 255, and write it out in the text format above. The **fixed
(untrained) weights** will still not diagnose anything — see `THEORY.md` §7 for
what a trained EfficientNet/RETFound model changes (the weights, not the
arithmetic). **Not for clinical use** (CLAUDE.md §1, §8).

## Provenance & field meanings

| Token | Meaning | Units |
|---|---|---|
| `C` | channel count (always 3: R, G, B) | count |
| `H`, `W` | image height, width | pixels |
| `label` | ground-truth DR grade 0–4, or `-1` if unknown (our sample is `-1`) | grade |
| pixel | colour intensity | dimensionless, `[0,1]` |

Synthetic data is labelled synthetic everywhere it appears; no clinical validity
is claimed or implied.

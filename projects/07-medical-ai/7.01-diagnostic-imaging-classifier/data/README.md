## Committed sample (`sample/`)

| Field | Value |
|---|---|
| File | `sample/imaging_sample.txt` |
| Origin | **Synthetic** — generated by `scripts/make_synthetic.py` |
| License | Public domain (CC0) — it is synthetic |
| Size | ~6 KB |
| Contents | The fixed model weights + a batch of 4 labeled 16×16 patches |

This tiny file lets `demo/run_demo` run **offline, with zero downloads**, a hard
requirement for every project. It is a plain whitespace-separated float file; the
parser is `load_sample()` in `src/reference_cpu.cpp`.

### Field-by-field layout

All values are floats (integers stored as floats and rounded on read). Geometry
constants come from `src/reference_cpu.h`: `IMG_H=IMG_W=16`, `NUM_F=4`, `KERNEL=3`,
`NUM_CLS=2`, `FLAT=196`.

| Order | Field | Count | Meaning |
|---|---|---|---|
| 1 | `n` | 1 | number of images in the batch (here 4) |
| 2 | `conv_w` | `NUM_F*KERNEL*KERNEL` = 36 | conv filter taps, row-major per filter |
| 3 | `conv_b` | `NUM_F` = 4 | one bias per filter |
| 4 | `dense_w` | `NUM_CLS*FLAT` = 392 | dense weights, row-major per class |
| 5 | `dense_b` | `NUM_CLS` = 2 | one bias per class |
| 6.. | per image: `label` then `pixels` | `1 + IMG_H*IMG_W` = 257 each | `label` ∈ {0 normal, 1 lesion, -1 unknown}; `pixels` row-major in [0,1] |

The **weights** are a hand-designed "lesion detector": filter 0 is a
Laplacian/center-surround stencil that fires on a bright blob; the dense layer
rewards filter-0 pooled energy for class 1 (lesion) and penalizes it for class 0
(normal). The **images** are 2 patches with a centered bright 4×4 blob (label 1)
and 2 flat/gradient patches (label 0), so batch accuracy is a meaningful 4/4 and
the demo output is interpretable (PATTERNS.md §6).

Regenerate or resize:

```bash
python scripts/make_synthetic.py                 # rewrite the committed sample
python scripts/make_synthetic.py --out other.txt # write elsewhere
```

## Full dataset

There is **no real dataset committed** — the catalog datasets below all require
registration and their licenses forbid casual redistribution. `scripts/download_data.ps1`
/ `.sh` print the source URLs and access instructions and **never bypass
credentials**; they defer to `make_synthetic.py` for an offline stand-in.

Catalog dataset notes (verbatim), with access notes:

- **MIMIC-CXR** — 227,827 labelled chest X-ray studies with radiology reports,
  Beth Israel Deaconess (https://physionet.org/content/mimic-cxr/). *Credentialed:*
  requires PhysioNet credentialing + a signed data-use agreement.
- **CheXpert** — 224,316 chest X-rays, Stanford, 14 pathology labels
  (https://stanfordmlgroup.github.io/competitions/chexpert/). *Registration
  required; research-use license.*
- **LIDC-IDRI** — 1,018 CT lung-nodule cases with radiologist consensus annotations
  (https://wiki.cancerimagingarchive.net/display/Public/LIDC-IDRI). *TCIA;
  generally CC BY 3.0 but confirm per-collection terms.*
- **TCIA** — The Cancer Imaging Archive, multi-modal oncology imaging across dozens
  of curated collections (https://www.cancerimagingarchive.net/). *Per-collection
  licenses.*

To use real images, you would export each to grayscale, resize to `IMG_H×IMG_W`,
scale intensities to `[0,1]`, and write them in the layout above (with a trained
set of weights). That is outside this teaching demo's scope (see THEORY.md §7).

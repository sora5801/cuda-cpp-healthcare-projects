
### File format (parsed by `load_problem()` in `src/reference_cpu.cpp`)

```
line 1:            width height patch_radius search_radius sigma h
next <height> rows: the NOISY image  — <width> floats per row, values in [0,1]
next <height> rows: the CLEAN image  — <width> floats per row (ground truth)
```

| Field | Meaning | Units |
|---|---|---|
| `width`, `height` | image dimensions | pixels |
| `patch_radius` (R) | patch is `(2R+1)²` px used for similarity | pixels |
| `search_radius` (S) | search window is `(2S+1)²` candidate px | pixels |
| `sigma` (σ) | noise standard deviation used to build the sample | intensity `[0,1]` |
| `h` | NLM filter strength (default `1.2σ`) | intensity `[0,1]` |
| pixel values | grayscale intensity | `[0,1]` (0 = black, 1 = white) |

### What the phantom is (and why it is shaped that way)

The clean image is a **Shepp-Logan-flavoured toy**: a dark background (`0.15`), a
large bright disk (`0.75`), and a smaller darker square inset (`0.45`). It
deliberately contains **large flat regions** (where denoising should clearly raise
PSNR) and **both a curved and a straight edge** (where a naive blur would fail but
NLM should preserve them). The noisy image adds fixed-seed Gaussian noise of
standard deviation `sigma`, clamped to `[0,1]`. The clean image is included **only**
so the demo can measure PSNR — it never influences the denoising itself (in the
clinic there is no ground truth).

## Full dataset (real medical data)

The committed sample is synthetic on purpose: the real datasets below either forbid
redistribution or require registration, so `scripts/download_data.ps1` / `.sh`
**print instructions and links only** — they never bypass any credential gate
(CLAUDE.md §8).

- **2016 AAPM Low-Dose CT Grand Challenge** — https://www.aapm.org/grandchallenge/lowdosect/
  Quarter-dose / full-dose CT *pairs* (the canonical low-dose-CT denoising benchmark).
  Requires agreeing to the challenge data-use terms.
- **NLST (National Lung Screening Trial)** via **TCIA** — https://www.cancerimagingarchive.net/
  Large chest-CT cohort; access through TCIA with a data-use agreement.
- **Fluorescence Microscopy Noise Dataset** (for Noise2Void) — https://github.com/juglab/n2v
  Real noisy/clean microscopy for self-supervised denoising.
- **SIDD** (Smartphone Image Denoising Dataset) — image-domain real noise, useful for
  sanity-checking a denoiser outside the medical domain.

### Layout

A whitespace-separated text file:

```
line 1 :  <n> <voxel_angstrom>          # here: "16 2"
line 2 :  n^3 floats  = map A           # 4096 density values, C-order (z,y,x)
line 3 :  n^3 floats  = map B           # 4096 density values, same grid
```

Index convention (C-order, x fastest): the value at voxel `(x,y,z)` is at flat
position `((z*n) + y)*n + x`.

### What the two maps represent (and why they are interpretable)

Both maps share the same noise-free **"true" density** — a sum of a few 3-D
Gaussian *atoms* (blobs), the kind of feature a real electron-density map of a
small molecule shows. Then:

- **Map A** = true + small isotropic noise (a clean reconstruction).
- **Map B** = true + larger isotropic noise + **high-frequency jitter** (a
  checkerboard-modulated term that puts its power at the highest spatial
  frequency).

Because the high-frequency content disagrees while the low-frequency content
matches, the **FSC curve decays at high frequency** — giving a meaningful,
recoverable "resolution" (8.0 Å for this sample). This is the whole teaching
point: see `THEORY.md` and `demo/README.md`.

This tiny file lets `demo/run_demo` run **offline, with zero downloads**, which
is a hard requirement for every project (CLAUDE.md §8).

## Full / real datasets

Real electron-density maps and the metadata to validate them come from public
archives. `scripts/download_data.ps1` / `.sh` print up-to-date instructions and
links; they never bypass any registration.

- **EMDB — Electron Microscopy Data Bank** (https://www.ebi.ac.uk/emdb/):
  deposited cryo-EM maps in CCP4/MRC format, often with the two **half-maps** you
  need for a gold-standard (0.143) FSC. Open license; downloadable by accession.
- **PDB / RCSB structure factors** (https://www.rcsb.org): crystallographic
  `F_obs` reflections for computing 2Fo−Fc / Fo−Fc difference maps and R/R-free.
- **wwPDB OneDep validation pipeline** (https://deposit.wwpdb.org): the official
  validation reports (RSCC, FSC, clashscore, …) for deposited structures.
- **IUCr validation standards** (https://www.iucr.org): crystallographic
  validation references (verify the exact dataset URL before use).

To read real MRC/CCP4 maps you would parse the 1024-byte MRC header (grid size,
cell dimensions, axis order) and the float32 voxel block — libraries such as
**GEMMI**, **CCP4**, or **mrcfile** do this. This teaching project uses the plain
text format above so the loader stays a few transparent lines; wiring in an MRC
reader is one of the exercises in `README.md`.

## Provenance & honesty

The committed sample is **synthetic** and labeled synthetic everywhere it

### Why a synthetic restraint list is the right choice here

Real NOE restraint tables come from assigned NMR spectra (BMRB / PDB; see below).
They are large, and each entry carries its own terms of use, so they are not suitable
for a tiny committed demo asset. Instead we synthesise the restraints **from a known
target structure** (a canonical α-helix Cα trace). This makes the demo
*interpretable*: because a satisfying structure provably exists, simulated annealing
should drive replicas to a low-energy, all-restraints-satisfied conformation — and
the demo reports exactly that (best replica: 19/19 restraints satisfied).

### File format (parsed by `load_config` in `src/reference_cpu.cpp`)

```
line 1:  n_beads n_restraints bond_len k_bond k_noe
line 2:  n_replicas n_steps T_hot T_cold step_sigma base_seed
next n_restraints lines:  i j upper
```

| Field | Meaning | Units |
|---|---|---|
| `n_beads` | backbone Cα atoms in the chain | count |
| `n_restraints` | number of NOE distance restraints | count |
| `bond_len` | ideal Cα(i)–Cα(i+1) virtual-bond length | Å (≈ 3.8) |
| `k_bond` | bond-restraint force constant | energy · Å⁻² |
| `k_noe` | NOE-restraint force constant | energy · Å⁻² |
| `n_replicas` | independent SA trajectories in the ensemble | count |
| `n_steps` | Monte-Carlo steps per trajectory | count |
| `T_hot`, `T_cold` | start / end annealing temperature | energy units |
| `step_sigma` | std-dev of a Gaussian trial bead move | Å |
| `base_seed` | global RNG seed (replica r derives its stream from it) | integer |
| `i j upper` | NOE restraint: beads i, j must be ≤ `upper` apart | indices; Å |

The committed sample: 12 beads, 19 NOE restraints derived from an α-helix, 512
replicas × 4000 steps. Bonded neighbours (j = i+1) are *not* listed as NOEs — the
bond term handles them.

## Full / real datasets

There is no single "download me" file for this project; real NMR restraint data is
assembled per protein entry. `scripts/download_data.ps1` / `.sh` print the canonical
sources and **never bypass any registration** (CLAUDE.md §8):

- **BMRB — Biological Magnetic Resonance Bank** (https://bmrb.io): assigned chemical
  shifts and restraint lists for thousands of NMR entries.
- **PDB NMR-derived structures** (https://www.rcsb.org): deposited NMR model
  ensembles; many entries include the original restraint files.
- **RECOORD**: a uniformly *recalculated* set of NMR structures — useful as a
  refinement benchmark.
- **CASD-NMR**: blind automated structure-determination benchmarks.

To experiment at larger scale without any download, regenerate a bigger synthetic
problem, e.g. `python scripts/make_synthetic.py --n-beads 24 --replicas 4096`.

## Provenance & honesty

The committed data is **synthetic**, generated deterministically from a geometric
α-helix model — it is *not* a measured spectrum and *not* a real protein. It is
labelled synthetic here, in the generator, and in the program output. Nothing in this
project may be used for diagnosis, treatment, or any real structural-biology decision.

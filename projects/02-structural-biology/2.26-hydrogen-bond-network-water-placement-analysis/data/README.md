### File format

Whitespace-separated text, read by `load_dataset()` in `src/reference_cpu.cpp`:

```
nx ny nz                         # voxel counts along x, y, z
ox oy oz spacing                 # grid origin (min corner) + voxel edge, Angstrom
nframes waters_per_frame natoms  # trajectory + solute sizes
<natoms lines>:   x y z charge   # solute atoms: position (A) + partial charge (e)
<nframes*waters_per_frame lines>:  x y z   # water-oxygen positions (A), frame-major
```

Field meanings (units):

| Field | Meaning | Units |
|---|---|---|
| `nx,ny,nz` | voxel grid dimensions | count |
| `ox,oy,oz` | world coordinate of the grid's minimum corner | Angstrom |
| `spacing` | cubic voxel edge length | Angstrom |
| `nframes` | number of MD snapshots | count |
| `waters_per_frame` | water oxygens recorded per frame (fixed) | count |
| `natoms` | solute atom count | count |
| solute `x y z` | atom position | Angstrom |
| solute `charge` | atom partial charge | elementary charge `e` |
| water `x y z` | water-oxygen position | Angstrom |

### What the synthetic sample encodes (the known answer)

The generator plants **two ordered, caged waters** at voxels `(4,5,5)` and
`(6,4,4)`: each is occupied in **every** frame (so density `g ≈ 240`) and sits in a
tight octahedral cage of neutral atoms that strains it (a mildly *unfavorable*
energy). Both effects raise the GIST free energy `ΔG`, so these two voxels must
appear at **ranks 1–2** of the program's displaceability list — the answer the demo
recovers. The remaining waters are diffuse and bulk-like. See `THEORY.md` §6.

## Full / real datasets

The committed synthetic sample is sufficient for the demo. To study GIST on real
structures, the relevant public sources (from the catalog) are:

- **SAMPL water-placement challenges** — https://github.com/samplchallenges/SAMPL —
  blinded benchmarks for water placement and binding.
- **Explicit-solvent PDB structures** — https://www.rcsb.org — crystallographic
  waters and protein coordinates (free; respect the RCSB usage policy).
- **GIST reference systems** — T4 lysozyme L99A and FKBP12 are the standard
  validation targets in the GIST literature.
- **WaterMap validation sets** — Schrödinger (commercial; URL to be verified).

Using these requires a real MD trajectory (AMBER/GROMACS/OpenMM) and a real GIST
tool (cpptraj `gist`, GISTPP) — outside this teaching project's scope. The
`scripts/download_data.*` scripts print these pointers and never bypass any
registration or license. **Respect every license; do not commit redistributed
data** (CLAUDE.md §8). Nothing here is for clinical use.

## Committed sample (`sample/protein_sample.txt`)

A 24-residue synthetic protein with a designed buried hydrophobic "core stripe"
and an exposed polar "surface stripe". Two fully-buried hydrophobic anchors are
planted (Leu at position 7, Phe at position 13) so the saturation-mutagenesis
scan recovers a stable, recognisable signal: mutating those core anchors to a
charged residue or proline is the most destabilising change.

### File format

Plain ASCII text, consumed by `src/reference_cpu.cpp::load_protein`:

```
line 1 : <name>            a single-token label (e.g. synthetic_core_helix)
line 2 : <L>               residue count (a positive integer)
next L : <AA> <buried>     one-letter wild-type amino acid + burial fraction
```

- `<AA>` is one of the 20 standard one-letter codes `A R N D C Q E G H I L K M F P S T W Y V`.
- `<buried>` is a fraction in `[0,1]`: `1.0` = fully buried core, `0.0` = fully
  solvent-exposed surface. It is the single scalar **structural feature** our
  reduced model consumes in place of a learned per-residue embedding.

| Field | Units / range | Meaning |
|-------|---------------|---------|
| name | token | label printed in the report |
| L | integer > 0 | number of residues (scan has `L × 20` cells) |
| AA | one of 20 codes | wild-type residue at this position |
| buried | `[0,1]` | local burial fraction (1 = core, 0 = surface) |

## Real datasets (study material — fetch with `scripts/download_data.*`)

The download scripts **print instructions and links**; they do not bypass any
registration or license. Real ΔΔG study sets you can learn from:

- **Protherm / ProThermDB** — >25k experimental ΔΔG values for single mutations.
  <https://www.abren.net/protherm/> (see ProThermDB for the maintained successor).
- **Megascale stability dataset (Rocklin lab)** — ~2.5M cDNA-display proteolysis
  stability measurements. <https://github.com/Rocklin-Lab/cdna-display-proteolysis-datasets>
- **ProteinGym** — standardized substitution/indel benchmarks for variant-effect
  and stability models. <https://github.com/OATML-Markslab/ProteinGym>
- **S669** — a curated, low-bias single-mutation stability benchmark used to
  evaluate ΔΔG predictors (search the literature for the current canonical URL).

Each of these has its own license and citation requirements; respect them. To
turn a real structure into this project's input you would compute a per-residue
burial fraction (e.g. relative solvent accessibility from DSSP/`freesasa`) — see
`THEORY.md` "Where this sits in the real world".

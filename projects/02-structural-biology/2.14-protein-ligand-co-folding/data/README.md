# Data — 2.14 Protein-Ligand Co-Folding

## Committed sample (`sample/`)

| Field | Value |
|---|---|
| File | `sample/complex_sample.txt` |
| Origin | **Synthetic** (generated by `scripts/make_synthetic.py`) |
| License | Public domain (CC0) — it is synthetic, not a real structure |
| Size | < 1 KB |
| Contents | A toy protein-ligand "complex": 12 protein backbone tokens on a pocket rim + 5 ligand atom tokens, plus the diffusion schedule |

This tiny file lets `demo/run_demo` run **offline, with zero downloads**, which
is a hard requirement for every project (CLAUDE.md §8). It is **synthetic** — a
hand-built geometry with a *planted* native pose so the demo has a known answer
(the reverse diffusion must recover it). It is **not** a real protein, a real
ligand, or a real binding pose, and carries **no clinical meaning**.

### File format

```
line 1 :  n_protein n_ligand steps temp step_frac type_bias seed noise_scale
line k :  type x* y* z*        (one row per token; type 0 = protein, 1 = ligand)
```

Field meanings (all coordinates in arbitrary "Angstrom-like" units):

| Field | Meaning |
|---|---|
| `n_protein` | number of protein backbone (C-alpha) tokens (the first rows) |
| `n_ligand` | number of ligand heavy-atom tokens (the remaining rows) |
| `steps` | T: number of reverse-diffusion (denoising) steps |
| `temp` | attention bandwidth/temperature; smaller = sharper attention |
| `step_frac` | DDIM step fraction in (0,1] — how far each step moves toward `x0_hat` |
| `type_bias` | additive attention bonus when query & key share a type |
| `seed` | RNG seed for the deterministic initial noise (reproducibility) |
| `noise_scale` | std-dev of the Gaussian noise added to `x*` to make the start `x_T` |
| `type` | 0 = protein backbone token, 1 = ligand atom token |
| `x* y* z*` | the token's **native** (target) 3-D position — the planted answer |

Regenerate (or make a bigger system) with:

```
python scripts/make_synthetic.py                                  # the default sample
python scripts/make_synthetic.py --n-protein 24 --n-ligand 9 --steps 240
```

## Full dataset

The demo needs no external data. For real protein-ligand complexes (to study how
production co-folding is benchmarked), see `scripts/download_data.ps1` / `.sh`,
which print sources and licensing and **never bypass registration**:

> PoseBusters benchmark — 428 recently released PDB complexes
> (https://github.com/maabuu/posebusters); PDBbind v2020 (http://www.pdbbind.org.cn);
> Astex Diverse Set — 85 drug-like ligand complex structures (verify URL);
> CASF cross-docking benchmarks (http://www.pdbbind.org.cn/casf.php).

- **License:** PoseBusters tooling is MIT; the underlying PDB entries have their
  own terms (check per entry). PDBbind and Astex require **registration / a
  license acceptance** — respect it; we redistribute none of their data.
- **Why the committed sample is synthetic:** because those datasets cannot be
  redistributed freely, and because a tiny planted-answer toy makes the teaching
  point (reverse diffusion recovers the pose) verifiable offline.

## Provenance & honesty

Everything in `sample/` is **synthetic and labeled synthetic**. No field here is
derived from a real patient, protein, or assay, and nothing in this project may
be used for any diagnostic or therapeutic purpose (CLAUDE.md §8).

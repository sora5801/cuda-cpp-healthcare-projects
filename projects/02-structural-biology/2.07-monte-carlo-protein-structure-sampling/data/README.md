
### Field meanings (this project's reduced 2-D HP lattice model)

| Token | Meaning | Units / range |
|---|---|---|
| `n` | chain length (number of residues) | integer, `2 ≤ n ≤ 64` |
| `sweeps` | MC sweeps per replica (1 sweep = `n` move attempts) | positive integer |
| `n_replicas` | independent Monte Carlo walkers (one per GPU thread) | positive integer |
| `t_min` | coldest replica temperature (refines minima) | `> 0`, in units of the contact energy `eps` |
| `t_max` | hottest replica temperature (crosses barriers) | `≥ t_min` |
| `seed` | base RNG seed; replica `r` uses stream `(seed, r)` | unsigned integer |
| sequence | `n` characters from `{H, P}` | `H` = hydrophobic, `P` = polar |

The default sample sequence `HPHPPHHPHHPHHPPHPH` is a **synthetic** 18-residue HP
chain (10 H, 8 P) — *not* a real protein. It is engineered to be small yet to
fold into a compact core with multiple buried H–H contacts, so the demo's
reported "best energy" is a meaningful, reproducible number (PATTERNS.md §6).

## Full / real-world datasets (not required for the demo)

This reduced-scope teaching model uses the simple **HP lattice protein** (Lau &
Dill, 1989), so it does **not** read full-atom structures. A learner extending it
toward a real MC engine would pull from the catalog's datasets — all gated by
their own licenses/registration, which `scripts/download_data.*` only links to:

- **CASP** structure-prediction benchmarks — <https://predictioncenter.org>
- **PDB** experimental structures (folding benchmarks) — <https://www.rcsb.org>
- **Dunbrack** backbone-dependent rotamer library — <https://dunbrack.fccc.edu/bbdep2010/>
- **CAMEO** continuous benchmarking — <https://www.cameo3d.org>

These are 3-D coordinate / rotamer datasets consumed by Rosetta or OpenMM; mapping
them onto a full-atom GPU MC engine is described in `THEORY.md` ("Where this sits
in the real world"). **No clinical use** — this is study material only.

## Provenance & honesty

Everything in `sample/` is **synthetic and labeled synthetic**. No patient-derived
or licensed data is committed. The demo never implies clinical or predictive
validity for real proteins.

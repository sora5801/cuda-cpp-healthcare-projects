# Data — 2.11 Cryo-EM CTF Estimation

## The committed sample (synthetic)

`sample/micrograph_sample.txt` is a **tiny synthetic cryo-EM micrograph**, committed
so the demo runs offline with zero downloads.

| field | value |
|-------|-------|
| Type | **SYNTHETIC** (forward-modeled, not a real specimen) |
| Size | 96 × 96 pixels (~88 KB text) |
| Generator | [`../scripts/make_synthetic.py`](../scripts/make_synthetic.py) (seed 2112) |
| SHA-256 | `42e2c49a74e0d2d93cf4e5643c26c3df41195cd5e61bea8f5a42d4ae6d470d04` |
| Ground-truth defocus | **15000 Å (1.5 µm)** — embedded in the header so the demo can score recovery |

> **This data is synthetic.** It is generated by passing white random noise through a
> known contrast-transfer function and adding Gaussian detector noise, so its power
> spectrum shows genuine "Thon rings" at the chosen defocus. It is a demonstration of
> the CTF-estimation algorithm, **not** a real electron-microscopy experiment, and
> carries no clinical or scientific claim about any specimen.

### File format (read by `load_micrograph` in `../src/reference_cpu.h`)

```
line 1:  n  pixel_size  lambda  cs  amp_contrast  true_dz
body  :  n*n floats, row-major (pix[y*n + x])
```

- `n` — image side length (pixels; square).
- `pixel_size` — detector pixel size on the specimen (Å/pixel).
- `lambda` — relativistic electron wavelength (Å); ~0.0197 Å at 300 kV.
- `cs` — spherical aberration (Å); `2.7e7` Å = 2.7 mm (a typical instrument).
- `amp_contrast` — amplitude-contrast fraction in [0, 1] (≈0.10 for cryo).
- `true_dz` — ground-truth defocus (Å) for synthetic data; set to `-1` for real data
  (where the defocus is the unknown being estimated).

## How to regenerate it

```bash
python scripts/make_synthetic.py                       # the committed sample
python scripts/make_synthetic.py --n 256 --defocus 12000 --out data/sample/big.txt
```

`make_synthetic.py` is deterministic given `--seed`, so the committed file is
reproducible.

## Real datasets (provenance & licensing)

Production CTF estimation runs on real micrographs from the **EMPIAR** archive:

| Dataset | What it is | Link |
|---------|-----------|------|
| EMPIAR home | Public electron-microscopy image archive | <https://www.ebi.ac.uk/empiar/> |
| EMPIAR-10025 | β-galactosidase micrographs (classic CTF tutorial) | EMPIAR entry 10025 |
| EMPIAR-10064 | T20S proteasome micrographs (RELION tutorial) | EMPIAR entry 10064 |
| RELION docs | Tutorial datasets + processing walkthrough | <https://relion.readthedocs.io> |

These are **free** but **large** (tens of GB) and in binary **MRC** format, which needs
an MRC reader (e.g. Python `mrcfile`) to convert into this project's text format. EMPIAR
data is released for research reuse under each entry's stated terms — check the entry
page before redistribution. We do **not** commit any real data; the sample here is
synthetic. See [`../scripts/download_data.ps1`](../scripts/download_data.ps1) /
[`.sh`](../scripts/download_data.sh) for the pointers and the conversion recipe.

### Field meanings

| Symbol | Meaning | Units |
|---|---|---|
| `n_tilts` | number of projection images (tilt angles) | count |
| `n_det` | detector bins per projection | count |
| `ds` | detector bin spacing | world length units |
| `img` | reconstructed slice side length | pixels |
| `world_half` | slice covers `[-world_half, +world_half]²` | world length units |
| `tilt_deg` | signed tilt angle of a projection | degrees |
| `p_j` | recorded (drift-shifted) projection intensity at bin `j` | arbitrary density·length |

### How the sample is engineered to be *verifiable* (PATTERNS.md §6)

The synthetic specimen is a small **disc phantom** (a bright central disc plus a
few off-center inserts, one of them "cold"/negative). A disc's projection (Radon
transform) is **analytic** — a ray sees a chord of length `2·√(r² − dx²)` — so
the tilt series is exact and deterministic, no rasterization. Three choices make
the sample teach this project's ideas:

1. **Limited, irregular tilt range** `[−60°, +60°]` with two angles dropped:
   the directions outside that cone are the **missing wedge**, and the schedule
   is non-uniform like a real acquisition.
2. **A known per-projection drift** (a small integer-bin sawtooth in `[−3, +3]`):
   the program's alignment step recovers it (printed as `estimated shifts`), so
   you can check the alignment against ground truth.
3. **A bright central disc** so the reconstruction's maximum lands near the slice
   center — a one-glance sanity check.

Regenerate (or make a bigger problem) with:

```bash
python scripts/make_synthetic.py                       # the committed sample
python scripts/make_synthetic.py --maxtilt 70 --step 4 --det 257 --img 192
```

## Full / real datasets

This project ships **only** the synthetic sample above; real cryo-ET tilt series
are large (multi-GB) stacks that must be registered before reconstruction and are
not redistributed here. `scripts/download_data.ps1` / `.sh` print where to get
them and never bypass any registration. From the catalog:

- **EMPIAR** tilt-series archives — <https://www.ebi.ac.uk/empiar/>
  (e.g. **EMPIAR-10045**, in-situ ribosome tilt series).
- **EMDB** subtomogram averages — <https://www.ebi.ac.uk/emdb/>.
- **SHREC** cryo-ET benchmark (verify the current URL on the SHREC site).

These are released for research under their own terms; consult each entry's
license before redistributing. To run this teaching project on real data you
would first export each tilt as a 1-D detector row (one slice) in the text layout
above — the loader is intentionally simple so the geometry stays legible.

### Layout (one whitespace-separated line)

```
L  n_bins  U_barrier  sigma  q  V  D  dt  n_steps  n_ions  seed
```

| Field | Meaning | Units |
|---|---|---|
| `L`         | pore length along the permeation axis z | nm |
| `n_bins`    | number of z-bins for the occupancy histogram | count |
| `U_barrier` | height of the central potential-of-mean-force (PMF) barrier | kT |
| `sigma`     | width (std-dev) of the Gaussian PMF barrier | nm |
| `q`         | ion charge (`+1` cation e.g. K⁺/Na⁺, `-1` anion e.g. Cl⁻) | e |
| `V`         | applied transmembrane voltage (reduced `e·V`); `+V` ⇒ forward current | kT/e |
| `D`         | diffusion coefficient of the ion in the pore | nm²/step-unit |
| `dt`        | Brownian-dynamics time step | step-unit |
| `n_steps`   | BD steps per ion trajectory | count |
| `n_ions`    | number of independent ion trajectories (the parallel work) | count |
| `seed`      | base RNG seed (ion `i` uses the reproducible stream `(seed, i)`) | integer |

The committed values are `3 12 4 0.5 1 4 0.4 0.2 2000 256 12345`: a 3 nm pore
with a 4 kT central barrier, a positive cation under a forward voltage, 256 ions
each walked for 2000 steps. These are chosen so the demo result is *interpretable*
(clear forward current + barrier-depleted occupancy; see `demo/README.md`).

> **All units are reduced** (thermal energy kT = 1). This keeps the Boltzmann
> factors and the Langevin drift term clean for teaching; converting to SI
> (picoamps, milliseconds) is left as an exercise in `../README.md`.

## Full / real datasets

This is a **synthetic teaching model**, so no real data is required to run it.
To ground the model in real structures and electrophysiology, the catalog points
to these sources (fetch instructions in `scripts/download_data.ps1` / `.sh`):

- **PDB ion-channel structures** — <https://www.rcsb.org> (e.g. KcsA `1BL8`,
  Nav, Kv). Public, freely redistributable; gives the pore geometry.
- **MemProtMD** — membrane-protein MD trajectories,
  <https://memprotmd.bioch.ox.ac.uk>. Provides equilibrated channel-in-membrane
  systems you could run real applied-field MD on.
- **Channelpedia** — patch-clamp electrophysiology (the real conductances to
  validate against), <https://channelpedia.epfl.ch>.
- **GPCRdb** — for GPCR/ion-channel families, <https://gpcrdb.org>.

Respect each source's license. None of these are committed here; the download
scripts print URLs and instructions and never bypass any registration. A real
PMF table `U(z)` would come from umbrella-sampling those trajectories — this demo
substitutes a closed-form Gaussian barrier so it is fully self-contained.

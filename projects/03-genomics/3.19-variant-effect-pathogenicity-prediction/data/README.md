This tiny file lets `demo/run_demo` run **offline, with zero downloads**, a hard
requirement for every project (CLAUDE.md §8).

### Field meanings (per variant line)

| Token | Meaning |
|---|---|
| `pos` | a **synthetic** 1-based coordinate, used only as a stable row label. Not a real genomic position. |
| `REF` | the reference allele (A/C/G/T) at the variant locus = the centre base of `WINDOW`. |
| `ALT` | the alternate allele (A/C/G/T) substituted in to form the alternate window. |
| `WINDOW` | `window_width` (=21) letters: the **reference** context. The variant sits at the centre (index 10). The loader builds the alternate window by copying this and flipping only the centre to `ALT`. |

### Why this sample is engineered (and labelled synthetic)

The shipped fixed model (`src/reference_cpu.cpp::init_model`) plants two motifs: a
**deleterious** 5-mer `CAGCT` (creating it → strong positive delta) and a **protective**
5-mer `TATAT` (creating it → strong negative delta). `make_synthetic.py` crafts a few
variants whose `ALT` allele *completes* one of these motifs at the centre, so the demo's
"most pathogenic" ranking recovers exactly those variants — a built-in known answer
(PATTERNS.md §6). **None of this is biologically real or clinically meaningful** — the
model is untrained and the motifs are invented.

Regenerate / resize:

```bash
python scripts/make_synthetic.py              # the committed 12-variant sample
python scripts/make_synthetic.py --n 2000000  # a large batch for a real GPU win
```

## Full dataset

Real variant-effect work uses labelled, partly access-controlled resources. The
download scripts (`scripts/download_data.ps1` / `.sh`) **print instructions and links
only** — they never bypass any registration or license.

- **ClinVar** — expert-curated pathogenic/benign variant calls.
  https://www.ncbi.nlm.nih.gov/clinvar/ (public; the VCF is downloadable from NCBI's FTP).
- **gnomAD** — population allele frequencies and per-gene constraint scores.
  https://gnomad.broadinstitute.org/ (public; large files).
- **MaveDB** — deep mutational scanning (DMS) atlas of experimentally-measured variant
  effects. https://www.mavedb.org/ (public).
- **HGMD** — Human Gene Mutation Database. http://www.hgmd.cf.ac.uk/
  (**licensed/credentialed** — registration required; the script will NOT bypass it).

If you wire any of these in, respect every license: do **not** commit redistribution-
restricted data; keep the committed sample synthetic.

### File format (what the loader reads)

A FASTA-like text file. `#` lines are comments; `>name` starts a record; the lines
after it are the record's residues (only the 20 standard amino acids, upper or
lower case). The loader (`src/reference_cpu.cpp::load_database`) concatenates each
record's residues. **Record 0 is the family CONSENSUS** — `main.cu` builds the
profile HMM from it; records 1.. are the search **database**.

```
# comments...
>consensus
MKTAYIAKQRQISFVKSHFSRQLE      <- record 0: builds the profile (24 match columns)
>homolog
MDTAYIAKQRQVSFRKSHFSRQLE      <- a 3-substitution mutant: the planted answer
>decoy1
QFLPVAAIKYGTVMRKNASQWDHC      <- i.i.d. random residues: no family signal
... (decoy2 .. decoy6)
```

### Per-field meaning

- **`>consensus`** — the family signature. Each of its 24 residues becomes a
  **match column** that emits that residue with high probability (`P_HIT = 0.55`)
  and the other 19 residues with low probability. This is the teaching stand-in
  for "estimate a profile from a multiple-sequence alignment".
- **`>homolog`** — the consensus with 3 random substitutions (`make_synthetic.py
  --subs 3`), modeling evolutionary divergence. It should rank **#1**.
- **`>decoyN`** — independent uniform-random sequences of the same length. They
  carry no family signal and should score far below the homolog.

Constraints (from `src/phmm.h`): profile length `MAX_M = 64`, sequence length
`MAX_L = 256`, alphabet = the 20 standard amino acids in the order
`ACDEFGHIKLMNPQRSTVWY`.

### Regenerating / enlarging the sample

```bash
python scripts/make_synthetic.py                 # the committed default
python scripts/make_synthetic.py --decoys 64     # more decoys
python scripts/make_synthetic.py --subs 6        # a more diverged homolog
```

Everything is deterministic given the seed, so `demo/expected_output.txt` stays
stable for the default sample.

## Full dataset

Real profile-HMM search uses large, license-governed corpora. The download
scripts (`scripts/download_data.ps1` / `.sh`) print the URLs and conversion
instructions; they do **not** download or bypass any registration.

- **Pfam-A** — ~20 k protein-family profile HMMs.
  Source: <https://www.ebi.ac.uk/interpro/download/> (`Pfam-A.hmm.gz`).
  License: **CC0**. Extract a consensus with HMMER's `hmmemit -c` and use it as
  record 0 of a FASTA file in this project's format.
- **UniRef50** — clustered protein sequences to search.
  Source: <https://www.uniprot.org/help/uniref> (`uniref50.fasta.gz`).
  License: **CC BY 4.0**. These become the database records.
- **Rfam** — RNA family profiles. Source: <https://rfam.org/>.
- **JGI metagenome proteins** — environmental targets.
  Source: <https://genome.jgi.doe.gov/> (**registration required** — the script
  prints instructions only).

### How it is engineered (and why)

The string is uniform-random DNA with the motif **`ACGT`** planted every 10 bases
(`scripts/make_synthetic.py`, fixed seed `20260628`). The planting makes the
substring **`ACG` occur exactly 6 times**, so the FM-index backward-search demo
has a **known, eyeball-checkable answer** (PATTERNS.md §6). The `$` sentinel that
the loader appends is the smallest symbol, which guarantees all suffixes are
distinct and the suffix array is unique — so `SA[0]` is always the sentinel
suffix (`60` for this 60-base input). Regenerate or resize with:

```bash
python scripts/make_synthetic.py            # default n=60
python scripts/make_synthetic.py --n 4000   # a bigger synthetic problem
```

Synthetic data is **labeled synthetic everywhere** it appears; it is not a real
genome or patient sequence and implies no clinical validity.

## Full dataset

The real targets for SA/BWT/FM-index construction are whole genomes and read
collections — far too large to commit, and several require registration. The
download scripts (`scripts/download_data.ps1` / `.sh`) are **idempotent** and
**print instructions and links only**; they never bypass any credential gate.

| Dataset | URL | License / access | Notes |
|---|---|---|---|
| **GRCh38** human reference (~3.1 Gb) | https://www.ncbi.nlm.nih.gov/assembly/GCF_000001405.40/ | Public domain (US Gov) | The canonical 3 Gb BWT-construction target |
| **1000 Genomes** read collections | https://www.internationalgenome.org/data | Open (Fort Lauderdale terms) | Pan-read BWT over many samples |
| **NCBI RefSeq** complete microbial genomes | https://ftp.ncbi.nlm.nih.gov/refseq/ | Public domain | Small genomes, good for scaling tests |
| **Human Pangenome** sequences | https://humanpangenome.org/ | Open (consortium terms) | Pan-genome BWT (terabase-scale frontier) |

When wiring a real dataset, follow the idempotent pattern the scripts describe:
skip the download if the file already exists with the right checksum, print the
source URL + expected size + SHA256, and for credentialed sets print the
registration instructions only.

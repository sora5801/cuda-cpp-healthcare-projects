### File layout (parsed by `load_genotypes` in `src/reference_cpu.cpp`)

```
# comment lines (start with '#', ignored) ...
N M                              <- two integers: #individuals, #SNPs
pheno: y_0 y_1 ... y_{N-1}       <- the N phenotype values (one quantitative trait)
<dosage_0> ... <dosage_{M-1}>    <- individual 0's M genotypes, each in {0,1,2}
...                              <- (N such rows)
snp: <id> <causal_flag>          <- SNP 0 metadata: rs-like id, 1 if causal
...                              <- (M such rows)
```

### Per-field meaning

- **dosage** `g_ij ∈ {0,1,2}` — the additive count of minor (effect) alleles
  individual `i` carries at SNP `j`. Sampled as `Binomial(2, p_j)` from a
  per-SNP minor-allele frequency `p_j ∈ [0.05, 0.5]` (Hardy–Weinberg draw).
- **pheno** `y_i ∈ ℝ` — a synthetic quantitative trait. Built as
  `y_i = Σ_{causal j} β_j · z_ij + noise`, where `z_ij` is the standardized
  genotype and `β_j` is the injected effect of causal SNP `j`. The noise is
  `Normal(0, 0.4)`. This is why the causal SNPs are *recoverable*.
- **snp id** — an `rsNNNN`-style identifier (cosmetic).
- **causal_flag** — `1` for the 5 SNPs that truly drive the phenotype (the demo's
  ground truth), `0` otherwise. The program prints whether each top hit is causal.

Regenerate (or make a bigger synthetic cohort) with:

```bash
python scripts/make_synthetic.py                       # the committed defaults
python scripts/make_synthetic.py --n 2000 --m 5000     # larger synthetic problem
```

## Full dataset

Real GWAS cohorts are **controlled-access** and may **not** be redistributed, so
no real data is committed here. `scripts/download_data.ps1` / `.sh` print how to
obtain them legally (they never bypass registration). The catalog sources:

- **UK Biobank** — ~500 k individuals, ~800 k genotyped variants. Requires an
  approved research application. https://www.ukbiobank.ac.uk/
- **dbGaP** — controlled-access GWAS datasets; requires a Data Access Request.
  https://www.ncbi.nlm.nih.gov/gap/
- **GWAS Catalog** — curated *published associations* (summary statistics, openly
  available; useful to cross-check recovered hits). https://www.ebi.ac.uk/gwas/
- **gnomAD** — population allele-frequency / LD reference panels (open).
  https://gnomad.broadinstitute.org/

| Source | License / access | Redistributable? | Size (order) |
|---|---|---|---|
| UK Biobank | Approved application | **No** | ~TB |
| dbGaP | Data Access Request | **No** | varies |
| GWAS Catalog | Open (summary stats) | Yes (attribute) | ~GB |
| gnomAD | Open | Yes (attribute) | ~GB–TB |

A real loader would read **PLINK2** `.bed`/`.pgen` or **BGEN** genotype files
rather than this text format; see PLINK2 in the README's "Prior art".
